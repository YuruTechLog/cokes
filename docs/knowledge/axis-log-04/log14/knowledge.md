## KN01: 拡散モデルの動作原理

**概念:** 画像生成AIの処理過程

**構造:**
```
砂嵐（完全ノイズ）
→ プロンプト語彙がエネルギー地形に等高線を刻む（傾斜形成）
→ 逆拡散: 傾斜に沿ってエネルギー谷（正解）へ降りる（30-50ステップ）
→ 画像生成完了
```

**適用:** 画像生成AI入門記事の核心説明として使用可能

## KN02: LoRA=フィルター比喩

**概念:** LoRAの役割を外付けフィルターで説明

**構造:**
- ベースモデル = カメラ本体

**適用:** 技術用語を知らない読者向け説明として最適

## KN03: 黄金律の数理的根拠

**概念:** 「構造が先、雰囲気は最後」の数理的証明

**根拠:**
- 拡散モデルの前半ステップ: 大局的構造の形成
- 拡散モデルの後半ステップ: 局所的細部・雰囲気の形成
- プロンプト設計も同じ: 構造語彙（前半）→ 雰囲気語彙（後半）

**実装:** Prompt Schedulingで実現可能
**適用:** 「構造が先」という直感を数理で裏付ける最強の論拠
