# Knowledge — axis-log-04 / log18

## KN01: 共同検証スタンス（実証なし執筆の価値転換）

**概念:** 実証なしの弱点を読者参加型の強みに転換する執筆スタンス

**転換:**
- Before: 「体験談」= 実証前提 → 環境なしは書けない
- After: 「エンジニアリングの提案」= 論理・原理ベース → 実証なしでも書ける

**価値:**
- 弱点（実証なし）→ 強み（読者と一緒に検証する立場）
- 「私がやってみた」< 「一緒にやってみよう」

**適用:** 実証環境がない状態での記事執筆スタンスとして汎用的に使用可能

## KN02: 理論と実装の一致確認

**概念:** AXISの黄金律が実装レベルでも成立することの証拠

**事実:** LoRA差替え時に前のLoRAがVRAMから解放 = Layer1完了後にLayer3(LoRA)を読み込む設計と完全一致
**意義:** 「理論が実装を規定している」= 黄金律の実装レベルでの確認
**適用:** 「AXISは実装とも整合する」の証拠として記事・説明で引用可能
