# Knowledge — axis-log-04 / log04

## KN01: 露光メタファー（事前設計の必要性）

**概念:** フィルム現像で「事後修正不可」を示す比喩

**構造:**
```
フィルム（意味空間）+ 光（指示）→ 現像後（応答後）にズレ判明
→ 事後修正ではなく事前設計（軸語彙の選定）が唯一の解
```

**適用:** 「なぜ応答後に修正しても限界があるか」の説明。AXIS事前設計の必要性を語る際の比喩

## KN02: AXISのブルーオーシャン確認

**概念:** 既存フレームワークとAXISの非競合確認

**対比:**
- 既存フレームワーク（CLEAR等）: 形式・トーン・構造の指定（形式層）
- AXIS: 意味空間への直接介入（意味空間層）

**結論:** 同レイヤーの競合なし = ブルーオーシャン
**適用:** AXISの独自性を説明する根拠として記事・提案書で引用可能
