# Knowledge — axis-log-04 / log07

## KN01: おやつ分解モデル（AXIS入門演習）

**概念:** 日常例でAXIS5軸を練習する演習素材

**例:**「おやつ」を5軸で分解:
- Intent（目的）: 空腹 / 気分転換 / 口寂しい
- Abstraction（抽象度）: チョコ全般 / 特定ブランド
- Boundary（範囲）: 甘いもの限定 / 塩辛いもOK
- Priority（優先順位）: 価格 / 味 / カロリー
- Direction（方向）: 選ぶ / 除外する

**適用:** AXIS入門演習素材として即使用可能。抽象的な5軸を直感的に体験させる

## KN02: AIツール利用の責任論

**概念:** AIのズレに対する責任の所在整理

**構造:**
- AIの応答がズレる → AI側の構造的限界（仕様）
- 設計者がそれを知らずに使う → 設計者の設計ミス
- 設計者がAXISで対処する → 責任を果たした設計

**適用:** AIツール利用の責任論・倫理論を扱う記事で参照可能
