## KN01: mp3メタファー（AI仕組み入門）

**概念:** AI内部ベクトルを音声デジタル化と同じ原理で説明

**構造:**
```
mp3: 音波 → 時間軸でサンプリング → 数値の羅列（波形データ）
AI:  意味 → 意味空間でサンプリング → 数値の羅列（ベクトルデータ）
→ 本質的に同じデジタル化の仕組み
```

**適用:** AI仕組み説明の入門として使用可能（専門知識不要）

## KN02: カラオケメタファー（ハルシネーション除去）

**概念:** ハルシネーション除去をカラオケのボーカル消しで説明

**構造:**
```
センターキャンセル（カラオケ）:
  センター帯域（ボーカル周波数）をカット → ボーカルが消える

ハルシネーション除去:
  特定意味帯域（不確実情報の周波数）をカット → 幻覚が消える
```

**適用:** ハルシネーション除去の直感的説明として最優秀。技術的背景なしで理解可能
