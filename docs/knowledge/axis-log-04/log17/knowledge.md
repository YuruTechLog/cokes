# Knowledge — axis-log-04 / log17

## KN01: 3レイヤー設計（画像生成の構造設計フレームワーク）

**概念:** 画像生成プロンプト設計の3段階レイヤー

**定義:**
- **Layer1 Skeleton**: 構図・ポーズ・大局的配置 → ControlNetで物理固定
- **Layer2 Material & Lighting**: 素材感・光源方向・影 → Prompt Scheduling前半配置
- **Layer3 Aesthetic Modulation**: 画風・雰囲気・スタイル → LoRA後付け / Prompt Scheduling後半

**適用:** 実践的画像生成プロンプト設計フレームワークとして即使用可能

## KN02: 「デッサンしてから色を塗れ」の3重一致

**概念:** 画家の直感・AI数理・Prompt設計の完全一致

**一致:**
- 画家の直感: デッサン（構造）→ 色塗り（雰囲気）
- AI数理: 前半ステップ（大局）→ 後半ステップ（細部）
- Prompt設計: 構造語彙 → 雰囲気語彙

**意義:** 3つが完全に一致 = 「世界初の言語化」として成立
**適用:** 記事タイトル・章見出し・SNSキャッチコピーとして最強候補

## KN03: Prompt Scheduling実装構文

**概念:** 黄金律をコードレベルで実装する構文

**構文:**
```
[構造語彙:雰囲気語彙:0.4]
```
→ 全ステップの前40%: 構造語彙が優先
→ 全ステップの後60%: 雰囲気語彙が優先

**適用:** 実装コード付きHow-to記事として使用可能
