# Knowledge — axis-log-04 / log02

## KN01: 食紅メタファー（AXIS動作説明の核心）

**概念:** 軸語彙が意味空間全体に浸透する動作の視覚的比喩

**構造:**
```
食紅（軸語彙）を水（意味空間）に投入
→ 全体に拡散・浸透（局所でなく全体が変化）
→ 水全体が染まる（応答方向が傾く）
```

**適用:** AXIS動作を直感的に説明する際の最優先比喩
**由来:** log02 食紅メタファー登場

## KN02: 日本語×AXISの構造的親和性

**概念:** 日本語の構造的特徴がAXIS必要性を高める

**根拠:**
- 主語省略 → AIが「誰が」を誤解
- 文末意味依存 → 文末まで読まないと意図不明
- 文脈前提 → 前提共有なしでAIは迷子

**適用:** 日本語対象の記事・説明で「なぜ日本人にAXISが特に重要か」の論点
**由来:** log02 日本語構造的弱点

## KN03: LLMの本質的課題（構造的欠陥ではなく仕様）

**概念:** AIは人間の意図を把握する仕組みをそもそも持たない

**重要な区別:** 「欠陥」ではなく「仕様」= 設計通りの動作
**含意:** 責任は設計者（指示する人間）にある

**適用:** AXIS導入の動機説明・AI活用責任論
**由来:** log02 LLM欠陥明確化
