## KN01: 重力場テンソルモデル（AXIS基盤比喩）

**概念:** AI意味空間を地形（重力場）として捉えるモデル

**構造:**
```
軸語彙 → 傾斜（引力の方向）を形成
指示   → 傾斜に沿ってエネルギー谷（正解）へ落ちる
応答   = エネルギー谷に到達した状態
```

**適用:** AXIS概念の説明・記事化・ハンドブック作成時の基盤比喩として常用
**由来:** axis-log-04 起源セッション

## KN02: CLEARとAXISの差異（レイヤー差）

**概念:** CLEARは形式指定・AXISは意味空間介入という根本的レイヤー差

**対比:**
- CLEAR: 何をどう書くか（形式層）
- AXIS: どこへ向かわせるか（意味空間層）

**適用:** AXIS紹介時に「既存フレームワークとの違い」を説明する際の核心論点
**由来:** log01 CLEAR比較
