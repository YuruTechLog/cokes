# Knowledge — axis-log-04 / log05

## KN01: 木構造モデル（AXIS視覚説明）

**概念:** 軸語彙の役割を木構造で説明する比喩

**構造:**
```
根（軸語彙）→ 幹（方向性）→ 枝（展開内容）
根が腐ると木全体が倒れる = 軸なし指示はどこへ行くか不明
```

**適用:** 視覚的説明資料・図解・スライドに使用可能

## KN02: 「人間把握なしAI処理不可」（コアメッセージ候補）

**概念:** AXISのキャッチコピーレベルのメッセージ

**論理:**
- 前提: AIは「何を求められているか」を自力で把握できない
- 必要: 人間が明示的に「把握情報」を注入する
- 手段: 軸語彙がその注入手段

**適用:** 記事タイトル・見出し・SNSキャッチコピーとして使用可能
