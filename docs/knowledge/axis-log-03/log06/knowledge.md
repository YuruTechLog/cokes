## KN01: チャンク粒度の定義（仮定）

**概念:** 適切なチャンクサイズの判定基準（初版）

**定義（仮）:** 「一つのチャンクで一つの判断が完結するサイズ」

**粒度のスペクトル:**
- 小（原子的）: 単語・短文レベル。統合コスト高・文脈維持困難
- 大（複合的）: 段落・セクションレベル。意味が拡散・不明確
- 適切: 「一判断=一チャンク」

**ステータス:** 仮定段階。log08で「完成度」として外部指標に接続  
**由来:** log06 チャンク粒度初言語化

## KN02: LLM地形崩壊 = 別安定点への収束

**概念:** ハルシネーションやおかしな出力は「ランダム」ではない

**構造:**
- 地形崩壊 = 通常の高ポテンシャル領域（正常応答エリア）から逸脱
- 落下先 = 別の安定点（谷）= 低ポテンシャル領域
- 谷では: 再現性のある一貫した（でも意図しない）出力が生成される

**実例:** nano banana（一貫した奇妙な絵の量産）  
**含意:** 同一異常出力が繰り返す場合、地形の「谷」を疑う  
**由来:** nano banana観察（log06）

## KN03: AI能力向上と暗黙知工学の需要

**概念:** AIが進化するほどCokesの需要が増大する理由

**メカニズム:**
- AI能力向上 → できることが増える
- 「何をAIに任せるか」の判断コストが増大
- 任せる/任せないの判断 = 必要十分条件 = 人間の暗黙知

**結論:** AI進化はCokesの需要を増大させる（逆相関しない）  
**由来:** 10年後予測議論
