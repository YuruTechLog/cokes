# Knowledge — axis-log-03 / log05

## KN01: おはじきモデル（離散チャンク意味空間）

**概念:** LLMの意味空間を離散チャンクとタグで表現するモデル

**構造:**
```
チャンク（意味の最小単位）
  ├─ タグ（チャンク間のリンク媒介）
  ├─ クラスタ（同一タグ群の集合）
  └─ 意味構築（クラスタ連結）
```

**Axis語彙との関係:** Axis語彙を与える = 特定タグ群を活性化 → そのタグを持つチャンクが集束 → 応答方向が決まる  
**重力場モデルとの関係:** 補完（重力場=連続体、おはじき=離散体）  
**由来:** log05 意味空間モデル議論

## KN02: 外部ツール使用前のAxis語彙必須ルール

**概念:** Axis語彙なしで外部ツール（LLM）を使うと汎用地形を走る

**運用ルール:**
1. 外部ツール（Gemini/GPT等）使用前にAxis語彙を準備する
2. プロンプト冒頭にAxis語彙を含めて意味地形を先にセットする
3. 語彙なしで使うと「デフォルト汎用回答」が生成される

**由来:** Gemini暴走インシデント（log05・07・11で3度発生）
