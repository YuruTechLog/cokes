## KN01: AI/人間レイヤー差分モデル

**概念:** AIと人間の処理レイヤーの構造的差分

```
人間
  ├─ 暗黙知層（体験・文脈・感覚・判断基準）
  └─ 言語化層 ──────────────┐
                              ▼
                           AI（言語のみ受取）
                              ├─ 言語処理
                              └─ 暗黙知なしで応答生成
```

**含意:** Cokesが暗黙知を言語化・構造化することで、AIが扱える状態にする  
**適用:** Cokesの価値を説明する際の基本図として使用  
**由来:** log04 レイヤー差分図議論

## KN02: Axis = 意味空間歪曲装置（確定）

**概念:** Axis語彙の動作メカニズム

**動作:**
- 旧モデル: 引力（磁石）→ AIを特定方向に引っ張る
- 確定モデル: 空間歪曲（ワープ）→ LLMの応答空間（意味地形）自体を変形する

**実用理解:** Axis語彙を与える = 地形の傾斜を変える = 応答が自然にその方向に流れる  
**由来:** 磁石→ワープモデル進化

## KN03: 重力場テンソルモデルの基本構造

**概念:** AI意味空間を重力場として捉えるモデル

**定義:**
- AI意味空間 = 多様体（曲がった高次元空間）上の重力場

- LLM応答 = 傾斜に沿って自然に流れる

**用途:** Axis動作の精密な説明。入門者には別比喩を先に使う
