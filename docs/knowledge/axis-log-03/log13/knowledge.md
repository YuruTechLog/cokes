# Knowledge — axis-log-03 / log13

## KN01: AI要約の根本的問題

**概念:** AIに長文を要約させると「骨格」が失われる

**失われるもの（骨格）:**
- 判断の連鎖（なぜAからBに至ったか）
- 前提の変化（途中で何が変わったか）
- 転換点（どこで議論が方向を変えたか）

**残るもの（外殻）:**
- 表層の言葉・結論のサマリー・キーワード

**根拠:** AIは言語パターンを圧縮するが、非言語の構造（思考の流れ・判断の変遷）は捨象する  
**由来:** AI要約=外殻のみ発見（log13）

## KN02: 骨格→AI統合ルートの設計

**概念:** 人間とAIの分業で最高品質の知識整理を実現する

**最強ルート:**
```
1. 人間が骨格を抽出
   （必要十分条件・判断基準・転換点・疑問点）
        ↓
2. AIが骨格に肉付け
   （統合・言語化・構造化・整形）
        ↓
3. 人間が確認
   （品質チェック・過剰品質除去・正確性検証）
```

**NG:** AIのみに要約させる（外殻のみ残る）  
**NG:** 人間のみで整理する（スケールしない）  
**由来:** 骨格→AI統合が最強ルートの確立（log13）
