## KN10: 海モデル語彙対応表

| 海の要素 | AIの対応概念 | 補足 |
|---------|------------|------|
| 海（全体） | AIモデル | GPT-4、Claudeなど |
| 海域 | 意味空間のエリア | 技術語の海域、詩語の海域 |
| 海水 | 意味空間そのもの | パラメータで性質が決まる |
| 塩分・温度 | パラメータの性質 | 学習データの偏り・重み |
| プランクトン | 概念の密度・頻度 | 高頻度語 = プランクトン豊富 |
| 表層 | 高頻度・日常概念 | 出力しやすい領域 |
| 深海 | 低頻度・高抽象概念 | 専門語・珍しい文脈 |
| 海流 | プロンプトの方向付け | ユーザーが起こす流れ |
| 濁り | ハルシネーション | 雑なプロンプトで発生 |

## KN11: ボトムアップ説明構造

```
1. 生活手触り（日常の体験）
2. 媒質・素材（片栗粉・水）
3. 意味空間（海のメタファー）
4. パラメータの複雑性（深海）
5. 再び生活手触りへ（「だからこうなる」）
```

**用途:** AI説明記事の章立て設計。「上から説明」より読者の理解経路に合う  
**由来:** log03「底から上ってきた構造」発見

## KN13: 「世界初」表現の安全クレーム

**使ってはいけない:** 「世界初の〜」  
**使える:** 「確認できる範囲で初の一般向け〜」「既存に類似例を発見していない〜」  
**理由:** 未翻訳文献・学術論文の先行例を完全否定できない
