## KN06: 媒質モデルの三分割定義

```
AI = 水槽（器・構造体）
意味空間 = 水（媒質・中身）
パラメータ = 水の性質（塩分・温度・密度等）
プロンプト = 性質を変える操作
```

**用途:** AIとプロンプトの関係を「操作」として説明するとき  
**注意:** 「流体モデル」と言うとき何が流体かを必ず明示する

## KN07: 片栗粉の国際化対応

| 日本語 | 英語 | 補足 |
|--------|------|------|
| 片栗粉 | cornstarch / cornflour | 素材名 |
| 片栗粉水溶液の固化現象 | oobleck / cornstarch slurry | 現象名 |
| ダイラタンシー | dilatancy / shear thickening | 学術名 |

**用途:** 英語圏向け記事・発表で比喩を使うとき

## KN08: 説明の対象層別モデル選択

| 対象層 | 推奨モデル | 理由 |
|--------|-----------|------|
| 高校生・一般 | 流体/媒質モデル | 生活手触り |
| 研究者・技術者 | エネルギー地形モデル | 物理的正確さ |
| 両層向け（記事2本構成） | 元記事（思想）＋深掘り（説明） | 独立完結 |

## KN09: 2段構造比喩設計（プロンプト強度別）

```
通常プロンプト → 流体モデル（媒質性質変化で方向制御）
強プロンプト   → ダイラタンシー（衝撃→硬化→反発）
方向制御       → 磁石モデル（引き寄せ・反発）
```

**注記:** 3モデルは独立ではなく強度帯別の状態遷移として機能
