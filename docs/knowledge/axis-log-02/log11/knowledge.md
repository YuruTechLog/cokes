# Knowledge — axis-log-02 / log11

## KN42: ゆるてく語の統合定義

**ゆるてく語 = 構造を保持しながら、軽く・一般的に扱える言語に変換する装置**

2つの側面（矛盾なし）:
- 「保持」= 暗黙知の構造・因果・操作の意味を失わない
- 「軽さ」= 専門語・深海語を重くなく読める表現に変換

**例:**
```
deep → 「local manifold activation」→「深海にピンポイントで光を当てる」
medium → 「shear thickening」→「叩くと固まる（片栗粉）」
light → 「Prior補完」→「知らないから学習バイアスで埋めた」
```

## KN43: Axis三層構造（最終整理）

```
思想体系（Cokes）
  ↑ なぜこの設計か / 根拠・思想的基盤
説明体系（海モデル・流体モデル）
  ↑ 一般向け説明ツール / 比喩
操作体系（Axis語彙）
  where / zoom / limit / weight / vector
  ↑ 実際に使う操作語
```

**使い方:** 相手が「何を求めているか」で層を選ぶ  
- 研究者 → 思想体系・学術語対応表  
- 一般ユーザー → 説明体系（海モデル）  
- 実践者 → 操作体系（Axis語彙）

## KN44: 整理屋・翻訳者としての自己定義

**Cokes/Axisの正確な自己定義:**
- ✗「世界を変える / 革命的発見」
- ✓「すでにある構造を形式化し、誰でも使えるようにする」

**役割:** 整理屋・翻訳者・形式化装置  
**価値:** 形式化によって「暗黙知→誰でも再現可能な知識」に変換  
**「中二病」回避:** 大言壮語ではなく「整理屋として機能すること」が誠実な自己説明
