# Knowledge — axis-log-02 / log08

## KN30: ゆるてく語の定義

**ゆるてく語 = 暗黙知構造を保持しつつ、一般的な言語に変換する翻訳装置**

- 保持するもの: 概念の構造・操作の意味・因果関係
- 変換するもの: 専門語→日常語 / 深海語→表層語
- 変換しないもの: 操作の意味・構造的正確さ

**例:**
- `local manifold activation` → 「深海の概念にピンポイントで光を当てる」
- `shear thickening fluid` → 「叩くと固まる流体（片栗粉）」
- `Prior補完` → 「構造がないので学習バイアスで埋めた」

**用途:** Axis・海モデルの概念を一般向け記事に翻訳するとき

## KN31: 構造渡しによるハルシネーション低減

```
構造なしプロンプト
  → LLM: Prior（学習バイアス）で補完
  → ハルシネーション発生

構造ありプロンプト（Axis的操作）
  → LLM: 指定された構造の範囲内で出力
  → ハルシネーション低減
```

**用途:** Axis操作の効果を説明するとき / Copilot等のツールの使い方説明に転用可

## KN32: 形式化の価値（Axisの独自性）

**「すでにやってる人がいる」≠「独自性がない」**

| 段階 | 内容 | 価値 |
|------|------|------|
| 発見 | 概念の発見 | 発見者の価値 |
| 暗黙実践 | 個人の操作 | 個人限定 |
| **形式化** | **言語化・体系化** | **誰でも再現可能** |

**Axisの価値 = 形式化の価値**
