# log11 Incident — axis-log-02

## I01: 「中二病みたいやな」の自己ツッコミへの対処

- 発生: 「Axis設計は中二病みたいで恥ずかしい」という自己ツッコミ
- 問題: 「世界を変える/再構築する」という大言と区別できていない
- 結果: 「世界再構築ではなく、すでにある構造を形式化するだけ」として切り分け
- 教訓: 「形式化・翻訳・整理屋」という自己定義が大げさな印象を防ぐ

## I02: ゆるてく語の定義の揺れ

- 発生: ログ08とlog11で「ゆるてく語」の定義が微妙に違う表現になった
- log08: 「暗黙知構造を保持しつつ一般化する翻訳装置」
- log11: 「構造を保ったまま軽く扱える言語」
- 結果: 同一概念の2つの側面。「保持」と「軽く扱う」は矛盾しない
- 教訓: 定義が複数表現になるときは「同一概念の別側面か、別概念か」を確認する
