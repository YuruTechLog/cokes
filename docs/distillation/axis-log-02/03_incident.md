# log03 Incident — axis-log-02

## I01: 水槽→海のスケールアップで意味が変わった

- 発生: 水槽モデルを「海」に拡張した時点
- 問題: 水槽は「閉じた系」、海は「開いた系」 → スケール変化で性質も変わる
- 結果: 海モデルでは「海域」「深海/表層」「プランクトン」等の新概念が必要
- 教訓: スケールアップは比喩の構造変化を伴う。再設計が必要

## I02: 「世界初」クレームの根拠薄弱リスク

- 発生: 海モデルを「世界初」と位置づけた時点
- 問題: 検索で見つからないだけで、学術論文・未翻訳記事に先行例がある可能性
- 結果: 「既存に確認できない」として留保表現を採用
- 教訓: 「世界初」は強い主張。「確認できる範囲で初」程度に抑える
