# log08 Incident — axis-log-02

## I01: Copilotハルシネーションの観察精度の問題

- 発生: Copilotが「構造渡されないとPriorで補完する」行動を観察
- 問題: 「これはCopilot固有か、全LLMに共通か」の判断が必要
- 結果: 「LLM全般の性質。構造渡しでハルシネーション低減」として一般化
- 教訓: 個別観察を一般化するとき「固有か共通か」を明示する

## I02: 「すでにやってる人がいる」と「発見の独自性」の緊張

- 発生: 「言語化してないだけですでにやってる人がいる」の発言
- 問題: Axisの独自性が「暗黙知を形式化しただけ」に見えるリスク
- 結果: 「形式化すること自体が価値。誰でも再現可能にすることが独自性」と整理
- 教訓: 「形式化」の価値は「発見」の価値とは別次元
