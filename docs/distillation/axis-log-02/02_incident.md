# log02 Incident — axis-log-02

## I01: 「流体」と「媒質」の概念混同

- 発生: 流体モデル採用後の精査段階
- 問題: 「AI＝流体」と「意味空間が流体（媒質）」が混在していた
- 結果: 「AI＝水槽」「意味空間＝媒質」「パラメータ＝媒質の性質」として三分割
- 教訓: 「何が流体なのか」を先に決めてから比喩を組む

## I02: 片栗粉の通文化的限界

- 発生: 英語圏向け記事展開検討時
- 問題: 「片栗粉」は日本語固有の語。海外では通じない
- 結果: oobleck / cornstarch slurry に翻訳
- 教訓: 生活素材系の比喩は文化依存度を事前確認する

## I03: エネルギー地形モデルの過剰スペック問題

- 発生: 流体より正確なモデルとして提案
- 問題: 「高校生でも読める」水準に合わない
- 結果: 保留（研究者向け説明には使用可）
- 教訓: 正確さとアクセシビリティはトレードオフ。対象層で切り替える
