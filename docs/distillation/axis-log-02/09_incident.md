# log09 Incident — axis-log-02

## I01: 数理抽象とSF考察の接続の飛躍リスク

- 発生: 「SF考察マニアが数理抽象に強い」という接続を行った時
- 問題: SF考察→数理抽象は「傾向」であり「因果」ではない
- 結果: 「SFを読めば数理抽象が鍛えられる」ではなく「系を内側から設計する習慣が共通」として整理
- 教訓: 相関から因果への飛躍に注意。「共通する能力の種類」として説明する

## I02: 「被弾しない」が攻撃回避に読める問題

- 発生: 「被弾しない 大事」の発言
- 問題: 「攻撃を避ける」「うまく逃げる」という解釈が生じる
- 結果: 「因果構造の事前理解による摩擦除去」として再定義
- 教訓: 比喩（被弾）は意図より広い意味を持つことがある。定義を添える
