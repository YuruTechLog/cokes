
## キー意思決定

| 決定内容 | 理由 |
|---------|------|
| SF考察=「系の内側から設計する習慣」として接続 | 直接因果ではなく「共通する能力の種類」 |
| 「被弾しない」=「因果構造の事前理解」として再定義 | 「攻撃回避」と読まれないための定義付与 |
| 感情ベクトル推測=人間固有 | AIは断片から会話方向を先読みできない |
