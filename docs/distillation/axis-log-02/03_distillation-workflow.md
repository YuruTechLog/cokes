# log03 Distillation Workflow — axis-log-02

## 処理フロー

```
log03.md
  ↓
[Phase1: Log Scan]
  - 海モデル設計（海域/深海/表層/塩分/プランクトン）
  - あらすじ生成（底から上ってきた構造）
  - CC-BY化判断
  - 世界初確認
  ↓
[Phase2: Distillation]
  主要トピック: T01〜T05 抽出
  インシデント: I01〜I02 抽出
  ↓
[Phase3: Summary]
  意思決定: 海モデル確立 / CC-BY公開 / 世界初留保表現
  ↓
[Phase4: Knowledge]
  再利用単位: 海モデル語彙対応表 / ボトムアップ説明構造
```

## キー意思決定

| 決定内容 | 理由 |
|---------|------|
| 海モデル確立 | 200Bパラメータ→海水性質で一般人理解可 |
| CC-BY化 | 普及優先。クレジット表記のみ条件 |
| 「世界初」→留保表現 | 検索外の先行例があり得るため |
| ボトムアップ説明採用 | 「底から上ってきた構造」が読者の理解経路と一致 |

- `03_log_topics.md` — トピック抽出
