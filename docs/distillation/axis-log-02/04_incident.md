# log04 Incident — axis-log-02

## I01: 「深海には絶対届かない」という誤仮定の修正

- 発生: 海モデル設計中「深海の概念は引き出せない」と暗黙的に想定
- 問題: 精度の高いプロンプト（ピンポイントの光）であれば深海も活性化可能
- 結果: 「条件付きで可能」に修正。「光の質次第」という新概念の追加
- 教訓: 「届かない」は「届かせ方を知らない」可能性がある

## I02: 学術用語との対応づけの難しさ

- 発生: 海モデルの章立てでAxis語彙と学術語を並列表示しようとした時
- 問題: 1対1対応ではなくN対Mの関係。どれを「代表」にするか不明
- 結果: 「Axis語彙は操作語、学術語は説明語」として役割を分離
- 教訓: 対応表は「等価」ではなく「機能的対応」として設計する
