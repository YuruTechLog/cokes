0. Overview — 海モデルとは何か
AI を「海」という媒質として捉える説明体系

一般人向けの直感モデル

Axis Framework の下層説明レイヤー

学術（ベクトル空間）との橋渡し

ハルシネーションを「濁り」として説明できる

1. 海底地形（学習データ）
● 海モデル
海底＝学習データの分布

深い谷＝強く学習された概念

浅瀬＝具体的知識

岩場＝偏った情報

砂地＝曖昧な領域

● Axis Framework との紐付け
Axis Words の「抽象度語彙」＝深海／表層の切り替え

Magnet Map の「効力差」＝地形の影響度

● 学術用語
Embedding space

Density of training data

Locality / manifold structure

2. 海水の性質（パラメータ）
● 海モデル

プランクトン＝ノイズ源

● Axis Framework との紐付け
Magnitude＝海水の重み付け

Filter＝濁りの除去

Constraint＝不可侵領域の設定

● 学術用語
Parameters

Activation functions

Regularization

Noise / stochasticity

3. 海流（プロンプト）
● 海モデル
プロンプト＝海流を起こす風

方向性語彙＝流れの方向

境界語彙＝流れの重さ

抽象度語彙＝深海／表層の切り替え

● Axis Framework との紐付け

● 学術用語
Prompt conditioning

Attention steering

Vector field manipulation

4. 出力（流れが最も伝わった方向）
● 海モデル
海流がもっとも伝わりやすい方向に出力が現れる

地形（学習データ）＋海水（パラメータ）＋流れ（プロンプト）の合成結果

● Axis Framework との紐付け
