# log06 Incident — axis-log-02

## I01: Bradの図が「実装理解」止まりであることへの誤解リスク

- 発生: Bradの公式図を「Axis/Cokesの根拠」として使おうとした時
- 問題: Bradの図はレイヤー設計の実務説明。「なぜこの構造か」の数理根拠ではない
- 結果: 「公式図はあくまで参照。Axis/Cokesの根拠はそれとは別」として切り分け
- 教訓: 公式図・権威資料は「実装事実の証拠」であり「概念根拠」ではない

## I02: 「人間負荷が増える」問題の未解決

- 発生: 「Axisは人間の負荷を増やすのでは」という懸念
- 問題: AIに合わせる（プロンプト試行錯誤）vsAxisを学ぶ（一定の学習コスト）
- 結果: 「合わせる疲弊」と「学ぶコスト」は性質が違う。Axis学習は一度で済む
- 教訓: トレードオフの「比較基準」を揃えてから議論する
