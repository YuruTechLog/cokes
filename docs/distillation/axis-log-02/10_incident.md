# log10 Incident — axis-log-02

## I01: 「ずぶとい」の意味の曖昧さ

- 発生: 「社会で敵を作る人はずぶとい」という表現使用時
- 問題: 「ずぶとい」が「神経が太い（ポジティブ）」とも読める
- 結果: 「感情ベクトルを読まない状態」として補足定義
- 教訓: 口語的表現は読み手によって意味が変わる。技術文書では補足定義が必要

## I02: セルフケアの設計が未完

- 発生: 「気にしすぎると胃に穴」の問題提起
- 問題: 「感度を持つ」と「疲弊しない」を同時に実現する設計が議論に上がったが未完結
- 結果: 「スイッチを切れる設計が必要」という方向性のみ確認
- 教訓: 問題提起で止まった議論は「要継続」として明示しておく
