## 主要テーマ構造

### Theme A: AI=1レイヤー上（暗黙知欠落）モデル
**構造**:
- 人間: 暗黙知（体験・文脈・感覚）→ 言語化 → AI入力
- AI: 言語のみ受取 → 暗黙知なしで応答生成

**含意**: Cokesが暗黙知を言語化・構造化することで、AIが扱える状態にする。これがCokesの存在意義。
