## 主要テーマ構造

### Theme A: おはじきモデル（離散チャンク）
**構造**:
1. チャンク: 意味のひとまとまり（離散単位）
2. タグ: チャンク間のリンク媒介
3. クラスタ: 同一タグ群の集合
4. 意味構築: クラスタが連結して意味を形成

**重力場テンソルモデルとの関係**: 重力場=連続体モデル、おはじき=離散体モデル。補完関係。

---

### Theme B: Axis語彙 = タグ群の操作
- Axis語彙を与える = 特定タグ群を活性化
- 活性化されたタグ群のチャンクが「引き寄せられる」
- 結果: 応答がAxis語彙に沿った方向に集束

---

### Theme C: チャンク粒度問題（初出・未解決）
- 粒度が大きすぎる: 意味が拡散・不明確
- 粒度が小さすぎる: 統合コスト増・文脈維持困難
- 適切な粒度の判定基準: 未定（log06で初言語化、log08で「完成度」として接続）

---

### Theme D: Gemini運用ルール
- Axis語彙なし → デフォルト地形（汎用回答）を走る
- **運用ルール確立**: Axis語彙を必須として与えてから外部ツール使用
