## 抽出した意味構造

### 構造1: Playwright工場の強度
```
再現性: 100回実行 = 100回同じ出力（CSSで制御）
メンテナンス性: CSS1行変更で全体デザイン変更
速度・コスト: 1秒未満・GPU不要・クラウドAPI不要
→ AIバナー生成の弱点（非決定性・文字化け・高コスト）を全て回避
```

### 構造2: ハイブリッドの正しい分業
```
AI（Stable Diffusion）: 背景テクスチャ・素材画像生成
                         → 多様性・センスが活きる部分
コード（Playwright）:    テキスト配置・レイアウト・最終合成
                         → 確実性・再現性が必要な部分
→ 「決定権はコードに」「素材だけAIに」
```

### 構造3: 「指揮する人」モデル
```
作る人:    全工程を自力でこなす（手作業・時間コスト大）
指揮する人: 各工程に最適なツールを配置・監督
  - 推論・分析: Claude Pro
  - 画像生成:   Stable Diffusion（必要な時のみ）
  - バナー合成: Playwright
→ 「AIを使うこと」が目的ではなく「結果を出すこと」が目的
```
