# Chat Summary — axis-log-04 / log19

## 概要
RTX5040ノートPCの適性確認。Ollama（LLM推論）vs Stable Diffusion（画像生成）のVRAM特性差の解明。「Claude Pro全投げ+ローカルがらがら」最強布陣確定。バナーのローカル移行推奨。

## 意思決定ログ（時系列）

### 1. RTX5040適性の確認
- 画像生成: 余裕（バッチ処理でVRAM解放が早い）
- LLM推論(Ollama): きつい（コンテキスト保持でVRAM継続消費）
- 「Ollamaできない=高い買い物」懸念 → 「画像生成専用戦車として最適」で解消

### 2. VRAMアーキテクチャの差異確認
- Ollama: ストリーム処理 → コンテキスト長×トークン分のVRAMを常時保持
- SD: バッチ処理 → ステップごとに前ステップを上書き解放

### 3. 最強布陣確定
- 推論・対話: Claude Pro（クラウド）
- 画像生成: RTX5040+SD（ローカル）
- ローカルGPUをLLMに使う必要なし

### 4. バナーローカル移行推奨
- DALL-E: 改変・再現性ゼロ・ブラックボックス
- ローカル: ControlNet（構図固定）+LoRA（画風後付）= 設計可能
