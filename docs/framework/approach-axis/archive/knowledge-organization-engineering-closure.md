# 記録: Knowledge Organization Engineering 検討の終結

- 起案: Fable 5（Advisor役）
- 起案日: 2026-07-10
- 終結日: 2026-07-23
- 状態: **不採用・アーカイブ**

## 結論

`Knowledge Organization Engineering` は、Cokesの一言定義、正式名称、内部コンポーネント名のいずれにも採用しない。

この語は、Cokesの定義を整理する途中で、暗黙知・判断・成果物を再利用可能な形にする働きを指すために用いた検討語であった。Cokes全体と同義ではないため、整理過程の記録としてここで終結する。

## 不採用の理由

- Cokesは、知識を整理するだけでなく、品質・人・コストの関係、責任、工程設計、監視、条件付きHandoffを一体として扱う。
- `Engineering System` とするCokesの内部に、さらに「知識整理工学」を置くと、何が上位概念で何を設計するのかが不明瞭になる。
- 知識整理は、Cokesを実装・運用するための中核機能であり、Cokesそのものの名前ではない。

## 今後の扱い

- Cokesの正本・用語集・説明文へ、この語を追加しない。
- 外部の知識組織化・知識表現・知識管理の考え方は、必要に応じて参考となる学術理論・実務知見として参照する。
- Cokes内では、機能を必要以上に固有名詞化せず、「知識整理」「判断記録」「成果物管理」として説明する。
