# 第5章: 軸語彙リファレンス（骨子）

> 本文ドラフト: `ch05_draft_v1.md`
> 用語の正本: `../glossary.md`

## 章の目的

軸語彙層で使う5軸と、AI特性層で参照する合金比率・磁石サイズを、実務で参照できる形にする。旧資料の「磁力強度マップ」や製造元別の観察は、固定的な法則ではなく条件依存の作業仮説として整理する。

## 構成

1. 5軸と補助軸（Sequence）の定義・例
2. 軸語彙の組み合わせ方と提示順
3. 軸語彙の効力傾向（旧「磁力強度マップ」）
   - 学術的な普遍則ではなく、Approach Axis独自の作業ヒューリスティクスとして明記
   - Sequenceの「補助」は効力の弱さではなく、提示・処理順を扱う機能上の区別として説明
4. 研究との対応範囲（直接根拠 / 間接根拠 / 独自整理）
5. AI Profileの4項目とモデル差の読み方
6. 合金比率（重みづけ思想）の観察
7. 磁石サイズ（チャンキング粒度）の観察
8. 実務での参照手順
9. 合金×粒度による失敗診断
10. 用語追加候補の提示

## 編集上の注意

- 軸語彙は磁石ではなく、意味地形に傾きをつける操作語彙として書く。
- 意味の配置・高低・近接関係は「意味地形図」。軸語彙の効力傾向と混同しない。
- モデル系列や製造元ごとの表は、実機検証を始めるための仮説表に留める。
- 論文・技術レポート・モデルカードは、参考となる学術理論・研究上の視点・公開資料として扱う。
- 新しい概念語は用語集へ直接追加せず、判断待ちの候補として提示する。
