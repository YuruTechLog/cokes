# 第3章: 3層構造の中での生息域

**目的:** 研究層を人間がイメージしやすい解釈層へ翻訳し、その結果を人間が使える操作層として実装したものがApproach Axisであることを示す。

## 骨子

- **研究層/解釈層/操作層の3層図を掲載**
  研究層（Activation Steering / SAE / Mechanistic Interpretability）→ 基板として存在。矛盾しないが使わなくても成立。
  解釈層（地形磁力モデル）→ 研究層の事実を人間がイメージしやすい形に翻訳する層。
  操作層（Approach Axis / 人間—AI IF）→ 軸語彙を選ぶ・プロンプトを設計する・出力を受け取り失敗を診断する。
  （参照: `drafts/terrain-magnetic-model.md` §研究層との関係）

- **参考となる研究上の視点を整理**
  Approach Axis概念（軸語彙が傾斜をつける / 地形崩壊=ハルシネーション / モデル別崩壊方向 / 操作判断言語）について、近接する研究領域と関係・限界を表にする。
  一致度の星評価や、特定時点の研究状況を網羅した主張は行わない。
  （参照: `drafts/ch03_draft_v1.md` §3.3）

- **操作層としてのApproach Axisを明示**
  Approach Axisは、研究層を直接実装するものではなく、解釈層のイメージを使って人間の意図・判断軸をAIへ渡し、AIの出力を人間が判定する操作層を設計する。
  研究上の知見の有無を網羅的に主張するのではなく、このプロジェクトが担当するIFの範囲を定義する。
  （参照: `drafts/terrain-magnetic-model.md` §研究層との関係 §結論）

> **[方針]** 外部参照は「参考となる学術理論・研究上の視点・公開資料」として扱う。Axisの直接的な証明や実装同一性は主張しない。
