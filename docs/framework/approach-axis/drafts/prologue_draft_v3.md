# プロローグ: AIの内側——意味の地形

> ドラフト v3 / 2026-07-11（仮）
> 位置: 序文の直後・第1章の前
> v2からの変更: ヒマラヤを地形説明の対比として明示。川=集落=出力の場所、ヒマラヤ=無人=到達困難の対比で密度差を表現。

---

AIの内部には、無数の意味のかけらが散らばっている。

学習で吸収した言葉、文章、概念——それらが分解されて積み重なり、広大な地形を作っている。

谷底には川が流れ、集落ができる。人が集まる場所だ。AIの意味空間でも同じことが起きている。よく使われる意味の組み合わせは低地に集まり、川になり、集落になる。あなたがプロンプト（AIへの入力文）を投げかけると、その言葉が磁石のように、地形上に散らばった意味のかけらを引き寄せる。
引き寄せられたかけらは重力に従い、川へ流れ落ちる。
そこで混ざり合い、出力になる。

対して、ヒマラヤ山頂には川も集落もない。「量子もつれの詩的な解釈」や「古代語の方言の語源」——そういう意味の組み合わせはあの絶壁の上にある。AIはそこへは滅多に辿り着かない。

何もしなければ、意味のかけらは「一番近い集落」に流れ着く。それがあなたの意図した集落かどうかは、関係ない。

---

ここで気づく人がいる。

地形は変えられない。AIが学習で作ったものだから。でも——**傾けられる**。磁石を当てるように、特定の方向に意味の流れを誘導できる。川の手前に傾斜をつけて、流れ着く先を変えられる。

さらに気づく。地形は**歪められる**。コンテキストを切り替えると、離れていた意味領域が突然近づく。紙を谷折りすると反対面の点が接触するように、地形の構造自体が変形する。

現実の地形は変えられない。でもここは違う。傾けられる。歪められる。

その認識を持った瞬間、あなたはもう測量者だ。

---

これはAI研究者が論文で扱う領域と同じ地図の上にある話だ。Activation Steeringと呼ばれる研究分野では、AIの内部に直接介入して意味空間の流れを変える実験が続いている。研究者はラボでそれをやっている。

Approach Axisはその地図を、プロンプトを設計する人の手に渡す試みだ。

地形の構造を読み、傾斜の向きを設計する——その道具。
