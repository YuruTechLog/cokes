# Axis Words — Layer 1

## 概要
軸語彙（Axis Words）は、AIの散乱ベクトルに外部から方向性を与える言語的ツール群。

## カテゴリ（最小セット）
- **起点（Intent）** : 「Aを起点に」「Aの観点で」
- **抽象度（Abstraction）** : 「概念レベルで」「具体レベルで」「抽象度を上げて」
- **境界（Boundary）** : 「この範囲だけで」「以下は除外して」
- **優先順位（Priority）** : 「最優先は○○」「○○＞△△」
- **方向性（Direction）** : 「〜方向に寄せて」「〜の観点で整列」
- **順序（Sequence）** : 「ステップ1：」「次に」「最後に」

## 使い方の原則
1. **明示的に軸を与える**（AIは自律で軸を作れない）  
2. **連番で順序を与える**（衝突を防ぐ）  
3. **境界で文脈を限定する**（探索空間を狭める）  
4. **抽象度を固定する**（粒度を安定化）  
5. **優先順位で評価軸を明示する**（判断基準を固定）

## 最小テンプレ（例）
- 「ステップ1：この観点で。ステップ2：この範囲だけで。ステップ3：抽象度を概念レベルにして。ステップ4：この方向に寄せて。ステップ5：最優先は○○。」

# Magnet Map — Layer 2

## 強度順（概観）
方向性（最強） > 境界 > 抽象度 > 起点 > 優先順位 > 順序（補助）

## 各カテゴリの効果
- **方向性（Direction）** : 探索方向を直接ねじ曲げる（最強）。  
- **境界（Boundary）** : attention 範囲を狭める。探索空間が減る（強）。  
- **抽象度（Abstraction）** : embedding の階層を固定する（中〜強）。  
- **起点（Intent）** : 目的方向を固定する（中）。  
- **優先順位（Priority）** : 評価軸を固定する（弱〜中）。  
- **順序（Sequence）** : 軸の衝突を防ぐ補助軸（補助）。

## 実務的示唆
- **まず方向性を与え、次に境界で絞る。**  
- **抽象度は早めに固定することで揺れを抑える。**  
- **優先順位はCopilot系で効きやすい。**

### CC‑BY が向いている理由（あなたのケース）

商用利用も OK（ただしクレジット必須）

学術系・note系・ブログ系と相性が良い

理論体系の普及に向いている

つまり、

「広く使ってほしいけど、著者としての名義は残したい」

### MIT が向いているケース

再利用自由

改変自由

商用自由

クレジット不要（ここが大きい）
